Przydatne Kwerendy PHP - INF.03

1. Deklaracja zmiennych

W PHP można deklarować zmienne za pomocą znaku dolara $. Przykłady:

// Deklaracja zmiennej
$zmienna1 = 'Hello, World!';

// Deklaracja zmiennej liczbowej
$zmienna2 = 42;

// Deklaracja zmiennej logicznej
$zmienna3 = true;

2. Funkcje

Funkcje w PHP mogą być deklarowane za pomocą słowa kluczowego function. Przykłady:

// Deklaracja funkcji
function dodaj($a, $b) {
    return $a + $b;
}

// Funkcja anonimowa przypisana do zmiennej
$odejmij = function($a, $b) {
    return $a - $b;
};

// Funkcja strzałkowa (PHP 7.4+)
$pomnoz = fn($a, $b) => $a * $b;

3. Pętle

PHP obsługuje różne rodzaje pętli. Przykłady:

// Pętla for
for ($i = 0; $i < 10; $i++) {
    echo $i;
}

// Pętla while
$j = 0;
while ($j < 10) {
    echo $j;
    $j++;
}

// Pętla do...while
$k = 0;
do {
    echo $k;
    $k++;
} while ($k < 10);

4. Warunki

Instrukcje warunkowe w PHP. Przykłady:

// Instrukcja if...else
$x = 10;
if ($x > 5) {
    echo 'x jest większe niż 5';
} else {
    echo 'x jest mniejsze lub równe 5';
}

// Instrukcja switch
$y = 'apple';
switch ($y) {
    case 'banana':
        echo 'To jest banan';
        break;
    case 'apple':
        echo 'To jest jabłko';
        break;
    default:
        echo 'Nieznany owoc';
}

5. Tablice

Praca z tablicami w PHP. Przykłady:

// Tworzenie tablicy
$tablica = [1, 2, 3, 4, 5];

// Dostęp do elementów tablicy
echo $tablica[0]; // 1

// Iteracja po tablicy
foreach ($tablica as $element) {
    echo $element;
}

6. Obiekty

Tworzenie i manipulacja obiektami w PHP. Przykłady:

// Tworzenie obiektu
class Osoba {
    public $imie;
    public $nazwisko;
    public $wiek;

    public function __construct($imie, $nazwisko, $wiek) {
        $this->imie = $imie;
        $this->nazwisko = $nazwisko;
        $this->wiek = $wiek;
    }
}

$osoba = new Osoba('Jan', 'Kowalski', 30);

// Dostęp do właściwości obiektu
echo $osoba->imie; // Jan

// Dodawanie nowej właściwości
$osoba->email = 'jan.kowalski@example.com';

7. Manipulacja DOM

Manipulacja elementami DOM za pomocą PHP. Przykłady:

// Pobieranie elementu
$element = $_GET['moj-element'];

// Zmiana tekstu elementu
$element = 'Nowy tekst';

// Dodawanie klasy do elementu
$element .= ' nowa-klasa';

8. Obsługa zdarzeń

Obsługa zdarzeń w PHP. Przykłady:

// Dodawanie nasłuchiwacza zdarzeń
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    echo 'Przycisk został kliknięty!';
}

9. Asynchroniczność

Praca z kodem asynchronicznym w PHP. Przykłady:

// Użycie funkcji sleep
sleep(2);
echo 'To jest opóźniona wiadomość';

// Użycie funkcji file_get_contents do pobierania danych
$data = file_get_contents('https://api.example.com/dane');
echo $data;

10. Formularze

Praca z formularzami w PHP. Przykłady:

// Pobieranie danych z formularza
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $dane = $_POST['nazwa-pola'];
    echo $dane;
}

11. Walidacja

Walidacja danych w PHP. Przykłady:

// Prosta walidacja formularza
$emailInput = $_POST['email'];
if (!filter_var($emailInput, FILTER_VALIDATE_EMAIL)) {
    echo 'Proszę podać poprawny adres email.';
} else {
    echo 'Adres email jest poprawny.';
}
